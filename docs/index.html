CPPS Launcher

A minimal, fast, and open-source launcher for Flash-based Club Penguin Private Servers.

This launcher uses Electron 11.5.0, optimized for compatibility with legacy Flash Player servers. It includes essential features while remaining incredibly lightweight and resource-friendly.

âœ¨ Features

  • Incredibly Lightweight & Fast Designed for minimal resource usage (Low CPU & RAM) and quick startup, focusing purely on running the game.
  • Native Windows Menu Uses the standard Windows menu bar, ensuring seamless integration with your system theme (Including Light/Dark mode).
  • Built-in Ad Blocking Blocks requests to known advertising and tracking domains at the network level, providing a cleaner experience.
  • Cosmetic Filtering Injects custom CSS to hide specific visual clutter and ad banners on certain servers (e.g., NewCP).
  • Crisp Flash Rendering Forces 100% display scaling (Disables OS scaling for the app) to prevent Flash content from appearing blurry on high-DPI monitors.
  • 32-Bit & 64-Bit Windows Support Automatically detects your system architecture and uses the appropriate included Flash Player plugin.
  • Discord Rich Presence Shows your activity as "Playing Club Penguin" in your Discord status while the launcher is running (Requires Discord desktop app).
  • Manual Update Check A simple menu option ("Options" -> "Check for Updates") that opens the project's latest release page on GitHub in your browser.
  • Clear Browsing & Flash Data An option to easily clear the application's cache, cookies, local storage, and importantly, the Flash Player's saved data (LSOs / "Flash Cookies") to resolve potential issues or reset game progress.
  • Fit Flash to Window Attempts to stretch the Flash game content to fill the entire launcher window using CSS injection. Results may vary depending on the game.
  • Zoom Controls Allows you to zoom the game view in, out, or reset it using standard keyboard shortcuts (Ctrl+Plus, Ctrl+Minus, Ctrl+0) or the Options menu.
